# Legit.ly Project Brief

## Product
Creator-owned storefront + universal affiliate marketplace for ecommerce brands.

## One-liner
For creators who want a real storefront and for DTC brands who want a self-serve affiliate engine without enterprise complexity.

## Target Users
- Shopify-first DTC brands ($1M–$20M revenue)
- Digital creators of any size (influencers, athletes, niche content creators, micro-ambassadors)

## The Wedge
A MySpace-inspired creator page that functions as a personal commerce site. Creators add affiliate products in one click. Brands get fast onboarding with trustworthy attribution and transparent payouts.

## Core Flows

### Creator: signup → profile → theme → marketplace → add products → publish page → share → track earnings → request payout
### Brand: signup → connect Shopify → import catalog → create offer → fund wallet → monitor → approve payouts
### Shopper: visit creator page → browse → click branded shortlink → buy on brand site → attribution tracked

## Attribution Model (MVP)
- Last-click, 7-day cookie window
- Coupon code fallback
- Shopify webhooks for order and refund sync
- Refund hold window before commission becomes payable
- Transparent ledger entries for every financial event

## Monetization
- Platform take rate on commissions paid (10–20%)
- Brand monthly subscription tiers
- Premium creator features later

## Launch Vertical
Sports merch, fitness, beauty, niche lifestyle DTC brands on Shopify.

## What This Is NOT
- Not a CRM
- Not a native checkout system
- Not a Linktree clone
- Not an enterprise affiliate suite
- Not a mobile app (mobile-optimized web only in MVP)
